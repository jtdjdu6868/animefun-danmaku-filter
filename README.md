過濾簡體字單純為了賞心悅目，並不代表簡體字就是偏激言論、也沒有特別針對族群、國家

# 功能
進階過濾器，可以過濾動畫瘋中的簡體字彈幕或是自訂無上限列表，可匯出匯入關鍵字
